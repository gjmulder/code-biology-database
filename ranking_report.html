About this report & caveats

Two independent axes over the same papers, reported side by side — neither is ground truth. Code-biology similarity = the per-chunk (8192-token "page") corpus-contrastive embedding score e per criterion, max-pooled over a paper's chunks (positive = reads, in the embedding space, like the domain-general code-biology poles — two independent worlds, a mediator, arbitrary rules — balanced across genetic, neural, semiotic and signalling instances rather than the genetic code alone; prototypes.json rev 2, test_runs.md Run 7). LLM verdicts are the domain-general graded judge's met / unclear / not_met per criterion (CLAUDE.md §9/§9.1), each met gated by a verbatim quote; papers not yet judged show –. Hover a verdict cell for its graded value and confidence. The verdicts are still synthetic labels from a comparatively weak judge, so ranks are more trustworthy than absolute magnitudes. Both axes are now domain-general yet corpus-wide ρ(e, verdict) stays flat, so they can still legitimately diverge paper-by-paper (test_runs.md Runs 6–7). The min metric is the weakest-link reading: per Barbieri a biological code requires all three criteria.
